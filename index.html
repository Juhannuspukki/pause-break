How to remap any key to pause break on Windows

The problem

So, here we are. You purchased Homeworld 2 Remastered but cannot use the tactical pause on HW2 Classic. Why? Because the pause function is mapped to the pause/break key which few keyboards have. Yours certainly does not.

Now what?

Solution that do not work so don't bother trying

You tried using the Windows on-screen keyboard to remap one of your Logitech mouse buttons to act as pause break. No cake. The command is interpreted as ctrl+insert and zooms in instead of pausing the game.

Then you try SharpKeys. Turns out that it supports all keys with the exception of - you guessed it - pause break. The scan code for pause break key is E1 1D 45 E1 9D C5 which is apparently exceptionally complex and cannot be slapped on to Windows registry like the other key codes.

Solution that works

The problem is solved by installing AutoHotkey and running the following script:

#Requires AutoHotkey v2.0

NumpadAdd::Send "{Pause}"

Those two lines are enough. After this the + button on numpad will act as pause.

If needed, the NumpadAdd can be replaced with any button on the keyboard. F7 would replace it with the F7 key, for example.
